# JournalsBR

Este é o repositório central de documentação do projeto JournalsBR, uma iniciativa do Centro de Pesquisa Política e Social (CPPS).

### Sobre o projeto

O **JournalsBR** começou como um repositório focado em agregar revistas de Relações Internacionais. Com o crescimento da equipe do Centro de Pesquisa Política e Social (CPPS) da UNESP de Franca, o escopo do projeto foi ampliado.
Nosso foco atual é criar um repositório de revistas de Direito, História, Serviço Social e Relações Internacionais.

### Objetivos

O objetivo central do projeto é contribuir para a universalização do conhecimento científico através da centralização e indexação de produções acadêmicas.

Identificamos que, frequentemente, pesquisadores enfrentam dificuldades para localizar artigos específicos em repositórios dispersos ou mesmo em indexadores generalistas, como o Google Acadêmico. Para solucionar esse problema, o JournalsBR não apenas agrega o conteúdo, mas realiza uma coleta estruturada de metadados. Isso nos permite desenvolver uma metodologia de indexação própria, desenhada especificamente para suprir as lacunas das ferramentas tradicionais e atender de forma mais precisa às necessidades de busca dos pesquisadores.

Atualmente, o projeto é um *pipeline* de coleta e organização de dados acadêmicos (PDFs, metadados e referências) dividido em cinco repositórios principais:

1.  Coletar Metadados
2.  Coletar PDFs
3.  Vincular Metadados e PDFs (XMP)
4.  RecollWeb
5.  Extrair referências dos PDFs
